# Git Core Commands

1. **git init** – команда инициализации репозитория

2. **git add .** – добавить все папки текущей директории в отслеживание изменений

3. **git commit -m "_somthing_"** – фиксирование изменения в локальном репозитории (**-m** _добавляет сообщение_)

4. **git status** – показывает текущее состояние репозитория

5. **git log** – выводит список коммитов

6. **git checkout _branchName || hashcode_** – переключает между ветками или восстанавливает файлы

7. **git branch** – показывает список всех локальных веток и отмечает текущую звёздочкой
    * **git branch _branchName_** – создает новую ветку
    * **git branch -d** – удаляет ветку
    * **git branch -m _newBranchName_** – переименовывает ветку

8. **git merge _branchName_** – слияние изменений из одной ветки в другую

9. **git push** – загружает изменения из локального репозитория в удалённый
